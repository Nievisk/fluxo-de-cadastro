# Sobre o projeto
Consiste na implementaçâo de um fluxo de autenticaçâo, incluindo backend, frontend, e banco de dados postgreSQL, escrito totalmente em TypeScript.
O objetivo a prática de fundamentos essensciais do desernvolvimento web, com foco prioritário em backend, mas integrando todas as camadas da aplicação para uma experiência mais realista.

#### Inclui
- Validação e tratamento de dados
- Modelagem de banco de dados
- Envio de e-mails
- Integração entre front e back
- Boas práticas de organização e padronização de código
- Orquestração de serviços com Docker


#### Técnologias
- TypeScript – Tipagem estática, segurança e padronização

- NestJS – Estrutura organizada para o backend

- Prisma ORM – Manipulação do banco PostgreSQL

- Docker + Docker Compose – Orquestração dos serviços

- JWT – Autenticação e gerenciamento de tokens

- Nodemailer (Gmail) – Envio de e-mails para validação

- React + TailwindCSS – Interface de usuário

#### Como executar
1: Abra o terminal e execute
```git clone https://github.com/Nievisk/login-flow.git```

2: suba os containers
```docker-compose up --build```

3: Use um cliente HTTP para testar o backend ou acesse a
[interface gráfica](http://localhost:5173).
